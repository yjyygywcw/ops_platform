Custom Page Calls

Page Templates are stored in 'pages' subdirectory, This function loads specific pages from that directory whenever you want. A simple funcion, but with extreme usage scenarios in an Admin Theme.

The sidebar works using the same technique

Usage:
loadContent('pagename');

For Example: To Load dummy.html from pages directory
loadContent('dummy');

In case of this Theme:
  • loadContent('dashboard');
  • loadContent('projects');
  • loadContent('messages');
  • loadContent('tasks');
  • loadContent('functions');
  • loadContent('settings');

Dynamically Add Projects

Here's another nifty tool to speed up development.

No need to push whole html through php, just parse the values in a json to this function.

Advantage: Consider a scenario say, for messages table, when size gets bigger, pushing whole html through php takes more time due to big size of html that has to be transferred, so how about a function where you can just past json and it updates html itself.

Usage: Go to the projects page and in developer console type this -
addProject({id:"4",name:"Brochure Design",client:"Wockhardt",desc:"Make biotechnology theme based brochure for Wockhardt",messages:"2",files:"2",tasks:"3",completion:"75%"});
INFO: The '+Add Project' Button in Project Page utilizes this function to add data to tables.


Dynamic Tasks Update

Managing Tasks is a very important aspect of Admin Frameworks.

Use these functions to update, add or remove tasks dynamically.

Usage:
tasks.add('Buy Beer for this Guy','0') //Add a new Task
tasks.remove('3'); //Removes the 3rd Task
tasks.updateState('2','0'); //Updates task with id=2 to completion state 0
